# Security Policy

## Reporting a Vulnerability

Please report security issues by opening a GitHub issue at https://github.com/0xs34n/summarizely-cli/issues.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |
